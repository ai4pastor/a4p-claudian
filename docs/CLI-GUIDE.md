# Claude/Codex 프로그램(CLI) 버전 안내

A4P Claudian이 "검증되지 않은 버전" 또는 "알려진 문제가 있는 버전" 경고를 보여줄 때 이 문서를 참고하세요.

## 왜 이런 경고가 나오나요?

Claude 프로그램(CLI)은 자주 업데이트됩니다. 새 버전이 플러그인과 아직 맞지 않는 경우가 가끔 있어서,
강사가 **검증을 마친 버전 범위**를 관리하고 있어요. 범위를 벗어나면 미리 알려 드리는 것뿐, 대부분은 그대로 써도 괜찮습니다.

## 지금 버전 확인하기

터미널(맥: ⌘+스페이스 → "터미널" / 윈도우: 시작 → "PowerShell")에 붙여넣기:

```
claude --version
```

## 검증된 버전으로 바꾸기 (다운그레이드)

경고 배너에 표시된 명령을 터미널에 붙여넣으면 됩니다. 예:

```
npm install -g @anthropic-ai/claude-code@2.1.211
```

> npm 방식이 아니라 설치 프로그램(네이티브)으로 설치하신 경우에는
> 경고 배너의 안내를 따르거나 강사에게 문의해 주세요.

## 최신 버전으로 업데이트하기

```
claude update
```

또는 npm 설치 사용자는:

```
npm install -g @anthropic-ai/claude-code@latest
```

## 문제가 계속되면

1. 옵시디언에서 명령어 팔레트(⌘/Ctrl+P) → **"환경 진단"** 실행 → 안내에 따라 조치
2. 그래도 안 되면 진단 화면을 캡처해서 강사에게 보내 주세요
